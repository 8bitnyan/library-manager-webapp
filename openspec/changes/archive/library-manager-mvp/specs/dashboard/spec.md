## ADDED Requirements

### Requirement: 대시보드 통계
시스템은 메인 페이지에 도서관 현황 통계를 표시해야 한다.

#### Scenario: 대시보드 통계 표시
- **WHEN** 인증된 사용자가 / 페이지에 접근
- **THEN** 총 도서 수, 현재 대출 중 수, 연체 수, 오늘 등록된 도서 수가 카드 형태로 표시됨

### Requirement: 최근 활동
시스템은 대시보드에 최근 대출/반납 활동을 표시해야 한다.

#### Scenario: 최근 활동 목록
- **WHEN** 대시보드 페이지 로딩
- **THEN** 최근 5건의 대출/반납 활동이 시간순으로 표시됨

### Requirement: 연체 현황
시스템은 대시보드에 현재 연체 중인 도서 목록을 표시해야 한다.

#### Scenario: 연체 도서 목록
- **WHEN** 연체 중인 대출 건이 존재
- **THEN** 연체 도서명, 대출자, 연체 일수가 목록으로 표시됨

#### Scenario: 연체 없음
- **WHEN** 연체 중인 대출 건이 없을 때
- **THEN** "연체 도서가 없습니다" 메시지 표시
