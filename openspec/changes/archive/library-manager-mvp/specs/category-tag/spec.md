## ADDED Requirements

### Requirement: 카테고리 CRUD
시스템은 도서 분류를 위한 카테고리를 관리할 수 있어야 한다. 카테고리는 단일 계층 구조(부모-자식)를 지원한다.

#### Scenario: 카테고리 생성
- **WHEN** 사용자가 카테고리 이름을 입력하고 생성
- **THEN** 카테고리가 생성되고 카테고리 목록에 표시됨

#### Scenario: 하위 카테고리 생성
- **WHEN** 사용자가 부모 카테고리를 선택하고 하위 카테고리 생성
- **THEN** 부모-자식 관계로 카테고리가 생성됨

#### Scenario: 카테고리 삭제
- **WHEN** 사용자가 도서가 연결되지 않은 카테고리 삭제 시도
- **THEN** 카테고리가 삭제됨

#### Scenario: 도서가 있는 카테고리 삭제 시도
- **WHEN** 사용자가 도서가 연결된 카테고리 삭제 시도
- **THEN** "해당 카테고리에 도서가 존재합니다" 오류 메시지 표시

### Requirement: 태그 관리
시스템은 도서에 자유 태그를 부여할 수 있어야 한다. 하나의 도서에 여러 태그를 연결할 수 있다.

#### Scenario: 도서에 태그 추가
- **WHEN** 사용자가 도서 등록/수정 시 태그를 입력
- **THEN** 태그가 도서에 연결되며, 존재하지 않는 태그는 자동 생성됨

#### Scenario: 태그 기반 필터링
- **WHEN** 사용자가 도서 목록에서 특정 태그를 선택
- **THEN** 해당 태그가 부여된 도서만 필터링되어 표시됨
