## ADDED Requirements

### Requirement: 도서 대출
시스템은 인증된 사용자가 도서를 대출할 수 있어야 한다. 대출 시 반납 기한은 14일이다.

#### Scenario: 도서 대출 신청
- **WHEN** 사용자가 대출 가능한 도서의 "대출" 버튼을 클릭
- **THEN** 대출 기록이 생성되고 반납 기한(14일 후)이 표시됨

#### Scenario: 이미 대출 중인 도서 대출 시도
- **WHEN** 사용자가 이미 대출 중인 도서의 대출을 시도
- **THEN** "이미 대출 중인 도서입니다" 오류 메시지 표시

### Requirement: 도서 반납
시스템은 대출 중인 도서를 반납 처리할 수 있어야 한다.

#### Scenario: 도서 반납
- **WHEN** 사용자가 대출 중인 도서의 "반납" 버튼을 클릭
- **THEN** 대출 상태가 "반납됨"으로 변경되고 반납일이 기록됨

### Requirement: 대출 이력 조회
시스템은 대출 현황 및 이력을 조회할 수 있어야 한다.

#### Scenario: 전체 대출 현황 조회
- **WHEN** 사용자가 /borrowings 페이지에 접근
- **THEN** 현재 대출 중인 목록과 반납 완료 이력이 표시됨

#### Scenario: 연체 도서 표시
- **WHEN** 반납 기한이 지난 대출 건이 존재
- **THEN** 해당 건이 "연체" 상태로 표시되고 시각적으로 강조됨
