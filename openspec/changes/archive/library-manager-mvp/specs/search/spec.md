## ADDED Requirements

### Requirement: 도서 검색
시스템은 제목, 저자, ISBN으로 도서를 검색할 수 있어야 한다. 검색은 부분 일치(LIKE)를 지원한다.

#### Scenario: 제목으로 검색
- **WHEN** 사용자가 검색창에 도서 제목의 일부를 입력
- **THEN** 제목에 해당 문자열이 포함된 도서 목록이 표시됨

#### Scenario: 저자로 검색
- **WHEN** 사용자가 검색창에 저자명의 일부를 입력
- **THEN** 저자명에 해당 문자열이 포함된 도서 목록이 표시됨

#### Scenario: 검색 결과 없음
- **WHEN** 사용자가 존재하지 않는 키워드로 검색
- **THEN** "검색 결과가 없습니다" 메시지가 표시됨

### Requirement: 카테고리 필터
시스템은 도서 목록을 카테고리별로 필터링할 수 있어야 한다.

#### Scenario: 카테고리 선택 필터링
- **WHEN** 사용자가 도서 목록에서 특정 카테고리를 선택
- **THEN** 해당 카테고리에 속한 도서만 표시됨

#### Scenario: 검색과 필터 동시 사용
- **WHEN** 사용자가 검색어 입력 후 카테고리 필터를 선택
- **THEN** 두 조건을 모두 만족하는 도서만 표시됨
