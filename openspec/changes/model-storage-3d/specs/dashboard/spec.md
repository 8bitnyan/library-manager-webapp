## MODIFIED Requirements

### Requirement: 대시보드 통계
시스템은 메인 페이지에 3D 모델 저장소 현황 통계를 표시해야 한다.

#### Scenario: 통계 표시
- **WHEN** 인증된 사용자가 / 접근
- **THEN** 총 모델 수, 카테고리 수, 총 파일 크기, 최근 7일 업로드 수가 카드로 표시됨

### Requirement: 최근 업로드
시스템은 최근 업로드된 모델 목록을 표시해야 한다.

#### Scenario: 최근 모델 목록
- **WHEN** 대시보드 로딩
- **THEN** 최근 5건의 모델이 이름/파일타입/등록일과 함께 표시

### Requirement: 카테고리별 분포
시스템은 카테고리별 모델 수를 차트로 표시해야 한다.

#### Scenario: 카테고리 차트
- **WHEN** 대시보드 로딩
- **THEN** 카테고리별 모델 수가 바 차트로 표시됨
