## MODIFIED Requirements

### Requirement: 모델 검색
시스템은 이름, 설명으로 모델을 검색할 수 있어야 한다. LIKE 부분 일치 지원.

#### Scenario: 이름으로 검색
- **WHEN** 사용자가 검색창에 모델 이름 입력
- **THEN** 이름에 해당 문자열이 포함된 모델 목록 표시

#### Scenario: 검색 결과 없음
- **WHEN** 존재하지 않는 키워드 검색
- **THEN** "검색 결과가 없습니다" 메시지 표시

### Requirement: 카테고리 필터
시스템은 모델 목록을 카테고리별로 필터링할 수 있어야 한다.

#### Scenario: 카테고리 + 검색 동시 사용
- **WHEN** 검색어 입력 후 카테고리 선택
- **THEN** 두 조건 모두 만족하는 모델만 표시
